terraform multiregion deployment
==================================


AWS VPC Network Design by SUZ-LAB

reference: http://blog.suz-lab.com/2012/11/vpc-suz-labvpc-cidr-azcidraz-cidr.html

- setup AWS Account
- create IAM INSTANCE PROFILE

|profile_name|
|------------|
|nat_profile|

- install terraform

- setup your terraform.tfvars

- execute terraform

```
$ terraform get
$ terraform plan
$ terraform apply
```
enjoy!
